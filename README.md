# نظام الكاشير المتكامل - Aysel & Asma System 🏪

برنامج كاشير احترافي متكامل لمحلات بيع الموبايل والكمبيوتر والإكسسوارات

## 🎯 المميزات الرئيسية

### 📊 إدارة المبيعات
- نقاط بيع (POS) متعددة
- فواتير مفصلة بالضرائب والخصومات
- طباعة فواتير + نسخ ديجيتال
- تعديل واسترجاع البضاعة
- أنواع دفع متعددة (نقد، كرت، شيك، تحويل بنكي)
- تقسيط للعملاء

### 📦 إدارة المخزون
- إضافة منتجات بالصور والأسعار
- تكويد البضاعة (Barcode/QR Code)
- تتبع الكميات والتنبيهات
- جرد المخزون الدوري
- تصنيفات متعددة (موبايل، كمبيوتر، إكسسوارات)
- أسعار مختلفة حسب الفئة

### 👥 إدارة الموردين والمشتريات
- قائمة الموردين الكاملة
- فاتورة الشراء من الموردين
- تتبع حالة الطلب
- مقارنة أسعار الموردين
- شروط دفع مختلفة
- تقييم الموردين

### 🛍️ إدارة العملاء
- قاعدة بيانات العملاء
- سجل المبيعات لكل عميل
- خصومات خاصة وعروض
- عملاء VIP
- نقاط الولاء والمكافآت
- تتبع الديون والمتأخرات

### 📈 التقارير والإحصائيات
- تقارير المبيعات (يومي/أسبوعي/شهري/سنوي)
- أكثر المنتجات مبيعاً
- أفضل العملاء والموردين
- المصروفات والأرباح
- تقارير الموظفين والحضور
- رسوم بيانية ولوحات تحكم

### 👨‍💼 إدارة الموارد البشرية
- إدارة الموظفين (كاشيرين، أمناء مخزن، مديرين)
- تتبع ساعات العمل والحضور
- الرواتب والعلاوات
- الإجازات والغيابات
- سجل الانضباط

### 🔐 الأمان والصلاحيات
- تسجيل دخول برقم سري قوي
- صلاحيات متعددة لكل موظف
- تتبع كل عملية (Audit Log)
- نسخ احتياطية تلقائية
- تشفير البيانات الحساسة
- استعادة البيانات

### ⚙️ الإعدادات المتقدمة
- إعدادات المحل (الاسم، العنوان، الهاتف)
- إعدادات الفاتورة والضرائب
- إعدادات الطباعة
- إعدادات المخزون
- إعدادات الموظفين والرواتب
- **إعدادات اللغة والمظهر**
  - اختيار اللغة (عربي/إنجليزي/فرنسي)
  - المظهر (فاتح/غامق)
  - الألوان والخطوط
  - التنسيقات (تاريخ/وقت/عملة)

### 🔧 ميزات إضافية
- دعم متعدد الفروع
- مزامنة المخزون بين الفروع
- طباعة الباركود والملصقات
- إرسال فواتير عبر البريد والـ SMS
- دعم متعدد العملات
- تطبيق محمول للموظفين والعملاء

## 🚀 المتطلبات والتثبيت

### المتطلبات
- Node.js 16+
- MySQL 5.7+
- npm أو yarn

### التثبيت السريع
```bash
# استنساخ المشروع
git clone https://github.com/xman888/aysel-asma-system.git
cd aysel-asma-system

# تثبيت الـ Backend
cd backend
npm install

# تثبيت الـ Frontend
cd ../frontend
npm install

# إعداد قاعدة البيانات
mysql -u root -p < ../database/schema.sql

# تشغيل الـ Backend
cd ../backend
npm start

# تشغيل الـ Frontend (من نافذة أخرى)
cd ../frontend
npm start
```

## 📊 قاعدة البيانات

**الجداول الرئيسية:**
- `users` - الموظفين والمديرين
- `products` - المنتجات
- `categories` - تصنيفات المنتجات
- `sales` - المبيعات
- `sales_items` - تفاصيل المبيعات
- `customers` - العملاء
- `suppliers` - الموردين
- `purchases` - المشتريات
- `inventory` - حركة المخزون
- `settings` - الإعدادات
- `reports` - التقارير

## 🔌 الـ APIs الرئيسية

### المبيعات
- `POST /api/sales` - إنشاء فاتورة
- `GET /api/sales` - قائمة المبيعات
- `GET /api/sales/:id` - فاتورة واحدة
- `PUT /api/sales/:id` - تعديل فاتورة
- `DELETE /api/sales/:id` - حذف فاتورة

### المنتجات
- `GET /api/products` - قائمة المنتجات
- `POST /api/products` - إضافة منتج
- `PUT /api/products/:id` - تعديل منتج
- `DELETE /api/products/:id` - حذف منتج

### الموردين
- `GET /api/suppliers` - قائمة الموردين
- `POST /api/suppliers` - إضافة مورد
- `PUT /api/suppliers/:id` - تعديل مورد
- `DELETE /api/suppliers/:id` - حذف مورد

### العملاء
- `GET /api/customers` - قائمة العملاء
- `POST /api/customers` - إضافة عميل
- `PUT /api/customers/:id` - تعديل عميل
- `DELETE /api/customers/:id` - حذف عميل

### الإعدادات
- `GET /api/settings` - الإعدادات الحالية
- `PUT /api/settings` - تحديث الإعدادات
